### Steps to re-create:
- Install Go from: https://golang.org/
- Create a directory structure:
	|_ go
		|_ src
			|_ github.com
				|_<user_name>
					|_ <project>
						|_ <package>
							|_ main.go
							
		|_ bin
		
- From the terminal: cd to the <package> folder
- Run: go run main.go
- To create an executable, run: go install
